# Repo-to-Knowledge

Built Repo-to-Knowledge, an agentic pipeline that extracts a codebase's structure via static analysis, uses the Gemini API to generate a knowledge base conforming to Google's Open Knowledge Format (released June 2026), cross-links it into a dependency graph, and exposes it through an MCP server queryable by any compatible AI host.
